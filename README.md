# Prediction-Black-Carbon
Prediccion de carbono negro por dia (1 - 7 dias) para el nevado Huaytapallana de la region de junin

# Conjuto de datos Necesarios:
 - Datos de carbono negro: https://www.datosabiertos.gob.pe/dataset/datos-de-carbono-negro-en-la-cordillera-huaytapalla-en-el-centro-de-monitoreo-de-glaciares-y
 - Datos exogenos: https://www.senamhi.gob.pe/?p=estaciones (Estacion: huaytapalla, periodos: 2022-05 -> 2024-04)
